# 🚀 Real-Time Chat Application

A **Real-Time Chat Application** that enables users to communicate instantly with each other using modern web technologies. This project demonstrates the implementation of real-time communication systems, including messaging, user interaction, and efficient data exchange.

---

## 📌 Overview

This project is designed to simulate real-world messaging platforms like WhatsApp or Messenger. It allows users to send and receive messages instantly with minimal delay using real-time communication protocols.

Real-time chat systems are widely used in modern applications for communication, collaboration, and customer interaction. ([Devpost - The home for hackathons][1])

## ⚙️ Installation & Setup

### 🔧 Prerequisites

* Java / Node.js installed
* Maven / npm installed
* Git

---

### 📥 Clone the Repository

```bash
git clone https://github.com/Ayush-kumar-01/Real-Time-Chat-Application.git
cd Real-Time-Chat-Application
```

---

### ▶️ Run the Project

#### If using Maven (Java project):

```bash
mvn clean install
mvn spring-boot:run
```

#### If using Node.js:

```bash
npm install
npm start
```
